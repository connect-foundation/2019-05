## Introduction

### **QuickKick은**

-   현재 아마추어(일반인) 팀이 풋살 한 게임을 하기 위해 투입해야 하는 불필요한 시간과 에너지를 줄여 아마추어 풋살 생태계를 활성화 시키고자 시작한 프로젝트입니다.

## Install

> client와 server 내에서 각각 실행해야 함.

```sh
yarn install
```

## Usage

```sh
yarn start
```
